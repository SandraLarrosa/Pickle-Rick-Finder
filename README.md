#### Instalación 🔧
Necesitarás instalar 📋 Node.js para trabajar con este repositorio.

##### Descarga o clona el repositorio. ⚙️
```bash 
$ git clone 'Direción del repo'
```
Abre el proyecto en tu editor de codigo.

Instala las dependencias locales:
```bash
$ npm install
```
Arranca el proyecto:
```bash
$ npm start
```
